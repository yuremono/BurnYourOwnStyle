# 1514 デザイン作成ドキュメント

作成日時: 2026-03-17 15:14

## 概要

pencil MCPツールを使用して、1920x3000ピクセルのベージュ色キャンバスと、中央に配置された1200x600ピクセルのピンク色四角形を作成しました。

## 実行したターミナルコマンド

### 1. 現在時刻の取得

```bash
date +"%H%M"
```

**出力:** `1514`

**解説:** 現在時刻を4桁の数字（HHMM形式）で取得し、ファイル名として使用します。

---

### 2. designディレクトリーの確認

```bash
ls -la designs/
```

**解説:** designディレクトリー内のファイル一覧を確認します。pencil-new.penが存在しないことを確認しました。

---

### 3. 空のpencil-new.penファイルを作成

```bash
touch designs/pencil-new.pen
```

**解説:** pencil MCPツールの`open_document("new")`は新規ドキュメントをメモリ上に作成するだけで、ファイルは自動的に作成されません。そのため、先に空のファイルを作成してから開く必要があります。

---

### 4. ファイルのリネーム

```bash
mv designs/pencil-new.pen designs/1514.pen
```

**解説:** デザイン作成完了後、pencil-new.penを現在時刻ベースのファイル名（1514.pen）にリネームします。

---

## pencil MCPツールで実行した操作

### 1. ドキュメントを開く

```
open_document(filePathOrTemplate: "designs/pencil-new.pen")
```

**解説:** 作成した空のファイルをpencilエディタで開きます。

---

### 2. デザインの作成

```javascript
// batch_design操作
canvas = I(document, {
  type: "frame",
  name: "1514",
  width: 1920,
  height: 3000,
  fill: "#F5F0E6",  // ベージュ色
  layout: "none",    // 絶対位置指定を有効化
  placeholder: true
})

pinkRect = I(canvas, {
  type: "rectangle",
  name: "Pink Rectangle",
  x: 360,            // (1920 - 1200) / 2 = 360 で中央寄せ
  y: 120,            // 上から120ピクセル
  width: 1200,
  height: 600,
  fill: "#FFB6C1"    // ピンク色
})

// placeholderフラグを削除
U("giz7k", {placeholder: false})
```

**解説:**
- **フレーム作成:** 1920x3000ピクセルのキャンバスを作成。`layout: "none"`を設定することで、子要素の絶対位置指定を可能にしています。
- **四角形作成:** 1200x600ピクセルのピンク色の四角形を作成。x座標は`(1920 - 1200) / 2 = 360`で中央寄せ、y座標は上から120ピクセルに設定。
- **色設定:**
  - ベージュ: `#F5F0E6`
  - ピンク: `#FFB6C1`

## 注意点

1. **pencil-new.penの自動作成:** `open_document("new")`を実行しても、実際のファイルは自動的に作成されません。先に`touch`コマンドで空のファイルを作成する必要があります。

2. **layout: "none"の重要性:** フレーム内で子要素の絶対位置（x, y）を指定する場合、親フレームの`layout`を`"none"`に設定する必要があります。デフォルトではflexboxレイアウトが使用され、x/yプロパティが無視されます。

3. **placeholderフラグ:** 作業中のフレームには`placeholder: true`を設定し、作業完了後に`false`に変更して完了をマークします。
